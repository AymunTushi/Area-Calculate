Q & A

Q1. What is DOM and purpose of DOM?

Ans: DOM stands for Document Object Model, which is a programming interface for web document. It allows to create, change and remove element from the document. We can create application that updates the data of the page without refreshing it.

Q2. What are the different ways to select DOM elements

  • getElementById - search element by element_id
  • getElementsByTagName - search element by tag name (e.g.,div,p)
  • getElementsByClassName - search element by class
  • querySelector - returns the first element that matches the specified selector

Q3. What is Event Bubble in JS?

Event Bubble is a type of event propagation. The event triggers the innermost target element and consecutively target its parent element. The innermost element's event is handled first then the outer.

Q4. What is Callback function and advantage of it?

A callback function is a function passed into another function as an argument. The callback function is helpful when we have to wait for a result that takes time. For example, the data coming from a server because it takes time for data to arrive
